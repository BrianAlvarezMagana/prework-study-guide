HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The A element stands for anchor used for hyperlinks
  • HTML acts like a frame of a house
  • HTML-> CSS-> JavaScript
  • Metadata is the data about our webpage that the browser uses to serve our webpage to users

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
  • Assigning colors in CSS can be done in many ways, including by semantic references such as gray or blue. We can also refer to a color by its hexadecimal code, or hex #

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to main branch
  • to sync the repo we must use these commands git add -A git commit -m “xyx” git pull origin main git push origin "Branch Name Here" git checkout main - after pushing on Git

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are a set of instructions to complete a task
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • For loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over
